# Cari Suku Kata e

A mobile-first React + TypeScript preschool revision game built with Vite.

## Run locally

```bash
npm install
npm run dev
```

## Features

- Parent and Child modes
- Parent syllable enable/disable settings
- Local progress using localStorage
- 10-question game session
- 3 answer cards per question
- SpeechSynthesis audio for instruction and feedback
- Stars, streak, simple celebration, and completion screen
- No login, backend, ads, links, chat, or in-app purchases
